⚙️ Cómo conectar Google Sheets (instrucciones para el programador)

Pasos para conectar el inventario real:

  1. Crea un Google Sheet con estas columnas exactas: marca | modelo | compatibilidad | stock | voltaje | procesador | imagen_url | descripcion
  2. Ve a Extensiones → Apps Script y despliega como Web App (o usa la API pública de Sheets)
  3. Reemplaza la variable SHEETS_URL en el JS con tu URL de despliegue
  4. Asegúrate de publicar el Sheet como "Cualquier persona puede ver"
  5. El inventario se actualiza automáticamente cada vez que alguien visita o hace clic en "Actualizar"

💡 URL de ejemplo: https://script.google.com/macros/s/TU_ID/exec
